# 星轨视界

中文 | [English](README.md)

一个基于 Web 的交互式 3D 太阳系可视化应用，支持离线运行，具有丰富的交互功能、动态拖影效果和中英文双语支持。

![太阳视图](images/sun.png)

![搜索视图](images/earch.png)

## 🌟 功能特性

### 核心功能
- **3D 太阳系渲染**：完整的太阳系，包含太阳和 8 大行星
- **交互式参考系切换**：点击任意行星或使用控制面板切换参考中心
- **动态拖影效果**：显示星体相对于参考中心的运动轨迹
- **行星信息面板**：点击行星后可查看详细信息
- **程序化纹理**：每个行星都有独特的纹理外观
- **真实光照系统**：太阳点光源 + 环境光，支持阴影
- **中英文双语支持**：自动检测浏览器语言，支持手动切换

### 控制功能
- **时间速度控制**：0x - 20x 可调节
- **拖影长度控制**：0 - 1000 可调节
- **拖影粗细控制**：0.1 - 5.0 可调节
- **显示选项**：可开关拖影、标签显示
- **视角预设**：俯视、前视、侧视、斜视、低空 5 种预设角度
- **暂停/继续**：可暂停动画

### 交互方式
- **鼠标拖拽**：旋转视角
- **滚轮**：缩放
- **点击行星**：切换参考中心
- **点击控制面板**：切换参考中心 + 调整相机角度
- **键盘快捷键**：
  - 空格：暂停/继续
  - 1-5：切换预设视角
  - R：重置视角
  - ESC：关闭面板/重置
  - +/-：调整速度
  - H：显示帮助

## 🚀 快速开始

### 前置要求
- 现代浏览器（Chrome、Firefox、Safari、Edge 等）
- 支持 ES Modules 和 WebGL

### 运行方式

#### 方式一：使用 Python HTTP 服务器
```bash
python3 -m http.server 8000
```
然后访问：http://localhost:8000

#### 方式二：使用 Node.js HTTP 服务器
```bash
npx http-server -p 8000
```
然后访问：http://localhost:8000

#### 方式三：使用启动脚本
```bash
./start-server.sh
```

### 离线使用
所有依赖已下载到本地 `libs/` 目录，完全支持离线运行，无需网络连接。

## 🎮 使用指南

### 切换参考中心
1. **点击场景中的行星**：立即切换到该行星为参考中心，相机保持不变
2. **点击控制面板按钮**：切换到该行星为参考中心 + 相机移动到俯视角度

### 观察拖影
- 切换参考中心后，所有星体会显示相对于该中心的运动轨迹
- 拖影长度根据星体距离自动调整（远星体拖影更长）
- 可通过滑块调节拖影长度和粗细

### 查看行星详情
1. 点击任意行星
2. 点击屏幕底部的"📋 显示详情"按钮
3. 查看行星的详细信息（距离、直径、公转周期等）

### 调整视角
- 使用控制面板的"视角预设"按钮
- 或使用键盘快捷键 1-5

### 语言切换
- 点击控制面板顶部的"EN/中文"按钮切换语言
- 页面加载时会自动检测浏览器语言
- 中文环境下自动显示中文，英文环境下显示英文
- 所有 UI 元素和行星标签都会实时切换语言

## 🔬 特殊功能

### 拖影系统
- 基于相对坐标计算
- 拖影长度根据星体距离自动调整
- 使用 TubeGeometry 实现可变粗细
- 切换参考中心时自动重新生成

### 相机系统
- 平滑动画过渡
- 支持多视角预设
- 聚焦时保持星体在中心
- 支持自由旋转和缩放

### 程序化纹理
每个行星都有独特的程序化生成纹理：
- **水星**：陨石坑纹理
- **金星**：噪点纹理
- **地球**：大陆纹理带云层
- **火星**：陨石坑纹理
- **木星**：风暴纹理带大红斑
- **土星**：条带纹理带光环
- **天王星**：噪点纹理
- **海王星**：噪点纹理

### 双语支持
- 基于浏览器语言设置自动检测
- 通过控制面板手动切换语言
- 所有 UI 元素动态切换语言
- 切换语言时行星标签实时更新

## 📝 注意事项

1. **浏览器兼容性**：需要支持 ES Modules 和 WebGL 的现代浏览器
2. **性能**：拖影数量较多时可能影响性能，可调整拖影长度
3. **离线使用**：所有依赖已本地化，支持完全离线运行
